# Линейная регрессия — California Housing

Предсказание медианной цены жилья в районах Калифорнии по 8 признакам: доход жителей, возраст домов, координаты и т.д.

## Датасет

`fetch_california_housing` из sklearn. 20640 объектов, 8 числовых признаков, без пропусков. Целевая переменная – `MedHouseVal`, медианная цена дома в районе в сотнях тысяч долларов.

## Стек

Python 3.12, pandas, scikit-learn, matplotlib.

## Что в ноутбуке

Разведка данных, гистограмма таргета, корреляции, scatter plot главного признака, train/test split 80/20, обучение `LinearRegression`, метрики MAE и R², коэффициенты модели, график предсказания vs реальность.

## Результаты

- MAE ≈ 0.53 (средняя ошибка ~$53k)
- R² ≈ 0.60

## Наблюдения по данным

Таргет обрезан сверху на 5.0 – около 1000 районов с реальной ценой выше $500k записаны как «$500k». Это особенность датасета. На графике предсказаний это видно как горизонтальная полоса точек ниже диагонали в правой части: модель не может предсказывать выше потолка.

Главный предиктор – `MedInc` (медианный доход), корреляция с ценой ~0.69. Координаты (`Latitude`, `Longitude`) связаны с ценой нелинейно, линейная модель этого не ловит.

## Что улучшит метрики

Масштабирование признаков, регуляризация (Ridge/Lasso), полиномиальные признаки, переход на нелинейные модели (Gradient Boosting).
